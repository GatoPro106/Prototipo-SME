# SME - Sistema de Monitoreo de Entorno

Proyecto de tesis desarrollado en el CECyT 18.

## Componentes

- ESP32-C3 SuperMini
- MAX 4466
- BME280
- MICS 5524
- Pantalla OLED

## Funciones

- Detección de gases.
- Medición de temperatura y humedad.
- Alarma por niveles peligrosos.
- Registro de datos.
